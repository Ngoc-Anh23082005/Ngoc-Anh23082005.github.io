# Portfolio Website - Ayla Pham

## Giới thiệu
Đây là website portfolio cá nhân được xây dựng bằng HTML và CSS, dùng để giới thiệu bản thân, các dự án đã thực hiện và định hướng nghề nghiệp trong lĩnh vực Software Testing.
Website được deploy bằng GitHub Pages.

## Công nghệ sử dụng
- HTML5
- CSS3
- Git & GitHub
- GitHub Pages
##Cấu trúc website
- **Home**: Giới thiệu bản thân và định hướng nghề nghiệp  
- **Projects**: Các dự án kiểm thử đã thực hiện  
- **Certificates**: Chứng chỉ và khóa học  
- **Contact**: Thông tin liên hệ
  
## Dự án tiêu biểu
### 1. QA Testing - Husol Vocab
- Kiểm thử ứng dụng học từ vựng tiếng Anh  
- Viết test case cho các chức năng học và ôn tập  
- Kiểm thử UI/UX  
- Phát hiện và báo cáo bug  
### 2. QA Testing - Website Bất Động Sản
- Kiểm thử website bán/cho thuê bất động sản  
- Test chức năng đăng ký, đăng nhập, tìm kiếm  
- Kiểm thử giao diện và trải nghiệm người dùng  
- Báo cáo lỗi  

## Định hướng Automation Testing
Hiện tại, tôi đang học và thực hành Automation Testing với Playwright, nhằm:
- Tự động hóa test case
- Kiểm thử giao diện và luồng chức năng
- Nâng cao hiệu quả kiểm thử
